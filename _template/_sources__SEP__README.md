# Sources

This folder holds source documents underlying the wiki's content.

## Structure

- **`raw/`** — permanent, read-only archive of ingested sources. See `raw/README.md`.

Additional subfolders may appear in projects with derivative source artifacts (transcripts, extracts, etc.); by convention, anything under `_sources/` is source material, not wiki content.

`canon_sources` citations on wiki pages point into `_sources/raw/`.
