# Obsidian Setup for Project Codex

## How to open this as Obsidian Vault

1. Open Obsidian
2. Click **Open folder as vault**
3. Select the root of this repository (`project-codex` folder)

**Recommended:**
- Use the whole repo as vault, or
- Use `documents/lattice/` as a focused vault

## Recommended Plugins
- Docxer (for .docx preview)
- HTML Reader
- Advanced Tables
- Dataview (very useful for Codex)

## Folder Structure
- `Sources/Raw/` → Original files (DOCX etc.)
- `0-Inbox/` → Quick capture
- `documents/lattice/` → Main knowledge base

Ready for Codex automation!
